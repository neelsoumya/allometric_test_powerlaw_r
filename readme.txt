readme

R script for performing SMA (semi-major axis) regression analysis of data
to test for allometric relationship.
We want to check for relationship X ~ Y^beta where beta is a scaling exponent
(log transformed logX ~ beta*logY + c)

1) rscript_sma_allometric_test.R
	R script (assumes X and Y variables are log transformed and stored in 
	separate cvs files)
